# Preparación de la Raspberry Pi 5

Esta sección detalla el hardware y software que necesitarás para comenzar con la instalación de Arch Linux en tu Raspberry Pi 5. Es crucial contar con los componentes adecuados para asegurar un proceso de instalación y funcionamiento sin problemas.

## Hardware Necesario
Asegúrate de tener los siguientes componentes antes de empezar:

*   **Raspberry Pi 5 (Model B):** El cerebro de nuestro proyecto. Asegúrate de que sea el Modelo B.
*   **Tarjeta microSD de buena calidad:** Mínimo 16GB. Se recomienda encarecidamente una tarjeta de 32GB o más, de Clase A1 o A2, para un rendimiento óptimo y espacio suficiente para el sistema operativo y las aplicaciones.
*   **Fuente de alimentación USB-C oficial para Raspberry Pi 5 (5V, 5A):** Es vital utilizar la fuente de alimentación oficial o una de especificaciones equivalentes (5V, 5A) para garantizar la estabilidad y el rendimiento de la Raspberry Pi 5, especialmente bajo carga.
*   **Cable HDMI a micro-HDMI:** Necesario si planeas conectar la Raspberry Pi 5 a un monitor o TV para la configuración inicial o para uso de escritorio. La Raspberry Pi 5 utiliza un conector micro-HDMI.
*   **Teclado y ratón USB:** Indispensables para la configuración inicial del sistema operativo si no se utiliza una conexión remota (SSH) desde el principio.
*   **Opcional: Carcasa para Raspberry Pi 5 con ventilación adecuada:** Altamente recomendada para proteger el dispositivo y, fundamentalmente, para mantener una temperatura de funcionamiento óptima. Considera una carcasa con ventilador activo o un buen disipador pasivo.
*   **Opcional: Cable Ethernet:** Útil para la conexión de red cableada inicial, especialmente si la configuración Wi-Fi no se realiza de inmediato. Proporciona una conexión más estable durante la instalación.

## Software Necesario
Para preparar la tarjeta microSD con la imagen del sistema operativo Arch Linux, necesitarás el siguiente software:

*   **Raspberry Pi Imager:** Es la herramienta oficial y recomendada por la Fundación Raspberry Pi para flashear imágenes de sistema operativo en tarjetas microSD de manera sencilla y segura. Está disponible para Windows, macOS y Linux.
    *   Puedes descargar Raspberry Pi Imager desde la página oficial: [https://www.raspberrypi.com/software/](https://www.raspberrypi.com/software/)
*   **Una computadora (Windows, macOS, o Linux):** Necesitarás una computadora para descargar la imagen de Arch Linux ARM (que obtendremos en la siguiente sección) y para ejecutar Raspberry Pi Imager, con el cual escribirás la imagen en la tarjeta microSD.

Con estos componentes de hardware y software listos, estarás preparado para pasar a la siguiente fase: la instalación de Arch Linux.
