Project "Surviving Chernarus" - Consolidated Development Plan Outlines (English)

Introduction: This document outlines the development steps for various interconnected projects under the "Surviving Chernarus" initiative. All service and functionality names have been translated to English for consistency in development, while the Wiki itself may remain in Spanish.

I. Top Priority: Core Personal Assistant and Themed Radio ☢️ * 1. Chernarus Vital Optimization Protocol (Personal Assistant) * Objective: Establish n8n as the core automation engine and integrate it with Google Calendar and Tasks for basic life planning, augmented by simple AI suggestions. * Original Spanish Name: El Protocolo de Optimización Vital de Chernarus * Key Steps: 1. n8n Setup (Raspberry Pi): * Install n8n (Docker recommended). * Secure the instance (credentials, HTTPS if/when external). 2. Google Calendar Integration: * Create Google API credentials (OAuth 2.0) with Calendar API scope. * Configure and authenticate the n8n Google Calendar node. * Test reading and creating calendar events. 3. Google Tasks Integration: * Ensure Google Tasks API is enabled for your Google Cloud Project. * Configure and authenticate the n8n Google Tasks node. * Test reading and creating tasks. 4. Basic Weekly Planning Workflow (n8n): * Trigger: Manual or cron-based (e.g., weekly). * Logic: * Fetch upcoming calendar events and current tasks. * AI Integration (Conceptual): Send data to an AI service with prompts to suggest tasks and outline a weekly schedule. * Initial Output: Send AI suggestions and plan to Telegram/email or a Google Doc. 5. Security Focus: Adhere to the principle of least privilege for API scopes; securely manage OAuth tokens.

*   **2. Radio Chernarus (Personalized Radio)**
    *   **Objective:** Set up a functional Icecast streaming server on the Raspberry Pi, with a basic n8n workflow to manage a playlist from local music files, and make the stream accessible externally.
    *   **Original Spanish Name:** Radio Chernarus
    *   **Key Steps:**
        1.  **Icecast Setup (Raspberry Pi):** Install `icecast2`, configure `icecast.xml`, enable and start service.
        2.  **Simple n8n-Orchestrated Streaming:** Organize music files. Install `ezstream`. n8n workflow to list files, create playlist, and run `ezstream`.
        3.  **External Access Configuration:** Set up DDNS, configure port forwarding on router, open firewall port on RPi.
        4.  **Initial Security Focus:** Strong Icecast passwords, consider restricting admin interface access.
