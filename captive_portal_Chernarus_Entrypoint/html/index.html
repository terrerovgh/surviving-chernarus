Welcome to the Chernarus Secure Network

To ensure secure and filtered internet access via Berezino_Checkpoint, you need to install our network's root CA certificate on your device.

Important: Without this certificate, you will experience security warnings and may not be able to access many websites (especially HTTPS sites).

Certificate Installation Guide

Please follow these steps:

  1. Download the CA Certificate:

    Download Chernarus_Root_CA.pem

    (If the download doesn't start, ensure `myCA.pem` is present in the `captive_portal_Chernarus_Entrypoint/html/` directory on the server and this link points to the correct filename.)

  2. Install the Certificate: The installation process varies by operating system:
    • Windows: Double-click the downloaded file, click "Install Certificate...", choose "Local Machine", click "Next", select "Place all certificates in the following store", click "Browse...", choose "Trusted Root Certification Authorities", click "OK", "Next", and "Finish".
    • macOS: Double-click the downloaded file. Keychain Access will open. Find the certificate (usually in 'login' keychain), double-click it, expand "Trust", and set "When using this certificate:" to "Always Trust".
    • iOS (iPhone/iPad): Download the certificate. Go to Settings > General > Profile (or VPN & Device Management). Tap on the downloaded profile and follow prompts to install. Then, go to Settings > General > About > Certificate Trust Settings and enable full trust for the root certificate.
    • Android: Download the certificate. Go to Settings > Security > Encryption & credentials (or Advanced > Encryption & credentials) > Install a certificate (or Install from storage). Choose "CA certificate" or "Wi-Fi certificate". You may be prompted to set a device PIN or password if you haven't already.
    • Linux: The process varies by distribution. Generally, you need to copy the `.pem` or `.crt` file to `/usr/local/share/ca-certificates/` and then run `sudo update-ca-certificates`.
  3. Reconnect to Wi-Fi: After installing the certificate, you might need to disconnect and reconnect to the `rpi` Wi-Fi network.

Once the certificate is installed and trusted, your internet traffic will be securely processed by `Berezino_Checkpoint`.
